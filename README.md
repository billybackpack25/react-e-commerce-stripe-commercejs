# Start 

1. Go into the project
    ```bash
    cd project
    ```

2. Install all the needed modules
    ```bash 
    npm install
    ```

3. Add a `.env` file
    ```javascript
    // .env file
    REACT_APP_CHEC_PUBLIC_KEY=...
    REACT_APP_STRIPE_PUB_KEY=...
    ```

4. Start up dev react
    ```bash
    npm start
    ```
